# Информационная система учета заявок пользователей

Проект моделирует небольшое консольное приложение для учета пользователей и их заявок.

Система позволяет:

- создавать пользователей;
- хранить сведения о пользователях в памяти приложения;
- создавать заявки от имени пользователей;
- получать список заявок;
- получать заявки конкретного пользователя;
- изменять статус заявки;
- деактивировать пользователей.

## Структура проекта

```text
src/
└── app/
    ├── models/
    ├── repositories/
    ├── services/
    ├── interface/
    ├── exceptions/
    ├── utils/
    └── main.py
```

Основные сущности приложения:

- `User` - пользователь системы;
- `Ticket` - заявка пользователя;
- `TicketStatus` - статус заявки.

Слой `repositories` отвечает за хранение данных в памяти. Слой `services` содержит основную бизнес-логику. Слой `interface` запускает демонстрационный консольный сценарий.

## Запуск

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите демонстрационный сценарий:

```bash
cd src
python -m app.main
```

Запустите тесты из корневой папки:

```bash
python -m pytest
```
